# Simple Greeting App

A simple Node.js + Express application that returns a personalized greeting message.

## Features

- Returns a greeting message via an API.
- Example: `http://localhost:3000/hello?name=Elvis`
- Default greeting if no name is provided.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/ElvisKemoi/simple-greeting-app.git
   ```
